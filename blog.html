What is the Document Object Model (DOM)?

In JavaScript DOM is a tree structure that is use to interact with HTML. The HTML file is the Document and all the elements of HTML are stored as Object, javascript create a tree structure model of this HTML element. So This is why its call DOM or Document Object Model. We can change or modify HTML elements using DOM

How do you select an HTML element from the DOM?

As we know that JavaScript see the html elements like object. So there are many
methods that we can use to select an HTML element. Like the

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • getElementByName()
  • querySelector()
  • querySelectorAll()

What is event delegation in the context of the dom, and why is it useful?

The event delegation is an efficient algorithm to access element.By following this algorithm we don't have to set event handlers to all the elements one by one, we can simply access the parent node and then we can use the event.target method to access the child elements. This way we can increase readability and efficiency of our code.

How do you manipulate an element's attribute and styles using DOM?

We can manipulate an element's attribute by using the set or getAttribute property and we can style an element by using classList.add or classList.remove function or we can directly add style in the element by using the template literal.
